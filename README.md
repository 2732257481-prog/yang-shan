# 杨珊 · 纪念网页设计方案

## 一、项目概述

一份以篮球场相识为起点的移动端纪念网页，用于向杨珊表达爱意与纪念你们的相识相恋历程。网页通过温暖的视觉语言和流畅的叙事节奏，将"相识→相伴→心声→照片回忆"的情感线索贯穿始终。

### 设计核心词
**温暖 · 篮球场 · 时光叙事 · 千年的温柔**

### 核心技术栈
| 技术 | 用途 |
|-----|------|
| HTML5 | 页面结构 |
| CSS3 (Flexbox/Grid) | 样式、动画、过渡、响应式布局 |
| Vanilla JavaScript | DOM 操作、触摸事件、滚动动画、音乐控制 |
| Intersection Observer API | 滚动驱动的元素渐入动画 |
| Canvas API | 粒子光晕效果 |
| Touch Events API | 照片滑动、缩放手势 |
| Pillow (Python) | 占位图片生成 |

## 二、整体设计理念

### 视觉灵感
网页以 **篮球场** 为视觉锚点——既是相识的地点，也是情感叙事的起点。暖橙色的黄昏球场光影贯穿始终，营造"日落时分，回忆铺开"的沉浸氛围。背景音乐选用 **"A Thousand Years" (Jada Facer & Kyson Facer 翻唱版)**，与你们从相识到相伴的历程天然呼应。

### 参考的优秀作品
| 参考作品 | 学习要点 | 融合方式 |
|---------|---------|---------|
| **Apple 产品页 (iPhone 15)** | 流畅的视差滚动 + 内容块优雅进出动画 | 页面采用 scroll-triggered reveal，每个板块进入时逐层淡入+上浮 |
| **One More Thing (Mac 个人页面)** | 极简留白 + 精致的字体排版 + 情感文字的力量 | 中文排版借鉴"大标题+小段落"的呼吸感节奏 |
| **Stripe Press** | 柔和色调 + 干净的卡片布局 | 暖橙 + 米白 + 淡粉的亮色调配色方案 |
| **Google Arts & Culture 故事线** | 图文交错的叙事节奏，触摸滑动的画廊交互 | 照片画廊采用横向滑动+描述文字叠层 |
| **Risu 纪念网页 (日韩风格)** | 轻柔的粒子动画 + 复古暖色调胶片感 | 首页篮球场背景加入微弱的粒子/光晕效果 |

### 配色方案
| 角色 | 色值 | 用途 |
|-----|------|------|

### 字体规范
- 标题：`Noto Serif SC`（思源宋体）, fallback `Georgia`
- 正文：`Noto Sans SC`（思源黑体）, fallback `system-ui`
- 字号规格：H1 2.4rem → H2 1.8rem → H3 1.3rem → Body 1rem

## 三、页面结构与交互流程

### 首页 (index.html)

#### 1. Hero 区
SVG 绘制的黄昏篮球场全景：暖色天空渐变 + 球场地面 + 篮筐剪影 + 夕阳余晖光线。前景通过 Canvas 粒子系统营造漂浮的金色光晕，模拟"回忆的尘埃"感。
- 大标题：**致 杨珊** + 英文副标题
- 副文案：*"从那个篮球场开始的故事，我想一直讲下去"*
- 按钮："开始翻阅" → 平滑滚动到相识板块
- 底部：滚动箭头提示动画

#### 2. 相识板块
以第一人称口吻讲述篮球场相识的故事。使用磨砂玻璃效果卡片，暖橙色装饰边框。

#### 3. 相伴板块
三个记忆卡片（打球·唱歌·玩游戏），带图标和短描述。使用 Intersection Observer 控制错峰入场动画。

#### 4. 心声板块
一段暖心的内心独白，文字逐行渐显。背景暖粉渐变。

#### 5. 照片画廊入口
圆形心形照片墙入口，点击后使用 morph 过渡跳转到 gallery.html。

#### 6. 页脚
深色背景的结束语 + "回到顶部"按钮。

### 照片展示页 (gallery.html)

#### 网格浏览
2 列缩略图网格，点击进入全屏浏览模式。

#### 全屏浏览
- 黑色半透明背景，照片居中
- **左右滑动**：Touch 事件驱动横向切换
- **双指缩放**：Pinch-to-zoom 手势（1x-3x）
- **双击缩放**：双击切换 1x / 2x
- **底部**：圆点指示器 + 照片标题 + 情感描述
- **关闭**：点击关闭按钮或按 ESC

## 四、交互与动效设计

| 交互点 | 效果 | 技术实现 |
|-------|------|---------|
| Hero 文字入场 | 逐层淡入 | CSS animation, 错峰 delay |
| 粒子光晕 | 持续漂浮 | Canvas requestAnimationFrame |
| 板块进入 | 淡入 + 上浮 40px | Intersection Observer + CSS transition |
| 卡片入场 | 错峰弹性上浮 | 不同 transition-delay |
| 心声文字 | 逐行渐显 | Intersection Observer 监听每行 |
| 照片切换 | 横向滑动 | touch events + translateX |
| 照片缩放 | 双指手势 | touch events + scale transform |
| 页面过渡 | 淡出/淡入 morph | CSS opacity transition |
| 音乐播放 | 浮动按钮 | fixed 定位 + Web Audio |
| 滚动进度 | 顶部细线进度 | scroll 事件 + width 百分比 |

### 音乐播放策略
- 用户首次点击页面后自动播放（绕过移动端自动播放限制）
- 右下角浮动播放/暂停按钮，带呼吸光晕动画
- 显示歌曲名 "A Thousand Years"
- 支持循环播放

## 六、照片素材使用指南

### 需要准备的照片表单（共 10 张）

| 编号 | 类别 | 内容建议 | 替换说明 |
|-----|------|---------|---------|
| 1-3 | 篮球场主题 | 球场合影、打球场景、球场黄昏 | 替换 `photo_01~03.jpg` |
| 4-6 | 日常合照 | 日常合影、自拍、约会照 | 替换 `photo_04~06.jpg` |
| 7-8 | 活动场景 | 唱歌、玩游戏、一起做事的时刻 | 替换 `photo_07~08.jpg` |
| 9-10 | 其他意义瞬间 | 礼物、纪念物、单人照 | 替换 `photo_09~10.jpg` |

### 照片技术要求
- **格式**：JPEG（或 WebP），最长边不超过 1200px
- **文件大小**：每张控制在 200KB 以内
- **命名规范**：保持 `photo_01.jpg` ~ `photo_10.jpg` 命名
- **描述修改**：编辑 `photos.json` 中的 title 和 description 字段

## 七、背景音乐设置

网页预设了 "A Thousand Years" 的背景音乐播放。请将 MP3 文件放置于：

```
assets/audio/a-thousand-years.mp3
```

**格式要求**：MP3, 128kbps 以内, 文件大小控制在 3MB 以内

如果使用其他歌曲，请同步修改 `assets/js/music.js` 中的 `this.audio.src` 路径。

## 八、部署方式

### 本地预览
直接使用浏览器打开 `index.html` 即可预览，无需本地服务器。

由于浏览器跨域安全策略，`photos.json` 的加载需要本地服务器环境：

```bash
# 使用 Python 启动本地服务器
python -m http.server 8080
# 然后在浏览器访问 http://localhost:8080
```

### 移动端预览
1. 确保电脑和手机在同一 Wi-Fi 网络
2. 启动本地服务器后，查找电脑的局域网 IP
3. 手机浏览器访问 `http://电脑IP:8080`

## 十、设计注意事项（已处理）

1. ✅ **移动端自动播放限制**：用户首次点击后触发播放
2. ✅ **安全区域适配**：使用 `safe-area-inset-*` 适配刘海屏和底部指示条
3. ✅ **字体加载优化**：`font-display: swap` + 预连接 Google Fonts
4. ✅ **图片懒加载**：`loading="lazy"` 属性
5. ✅ **触摸反馈**：`:active` 状态 + touch 反馈
6. ✅ **滑动冲突**：`touch-action` 控制
7. ✅ **Safari 100vh 问题**：使用 `100dvh` 方案
8. ✅ **GPU 加速动画**：使用 `transform` + `opacity`
9. ✅ **响应式设计**：360px ~ 768px+ 全面适配
